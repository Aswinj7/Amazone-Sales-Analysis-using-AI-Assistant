# 🤖 AI Data Assistant for Amazon Sales

An AI-powered analytics application that allows users to ask natural language questions and get insights from Amazon sales data.

---

## 🚀 Features

- 📊 Power BI Dashboard for visual insights
- 🗄️ MySQL database for structured data storage
- 🤖 OpenAI integration for Natural Language → SQL conversion
- 🐍 Python backend for query execution
- 🎨 Streamlit UI for interactive experience

---

## 🧠 How It Works

1. User asks a question
2. OpenAI converts it into SQL
3. SQL runs on MySQL database
4. Results are formatted and displayed

---

## 🛠️ Tech Stack

- Power BI
- MySQL
- Python
- OpenAI API
- Streamlit
