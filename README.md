# 🤖 AI Data Assistant for Amazon Sales & Customer Insights

An end-to-end AI-powered analytics system that allows users to interact with an Amazon Sales dashboard using natural language and instantly retrieve insights.

This project combines **Business Intelligence, Database Management, AI, and Web App Development** into a single workflow.

---

## 🚀 Project Overview

This project was built to solve a common problem:

👉 *"How can non-technical users query data without writing SQL?"*

The solution:
- Build a dashboard for visual insights
- Store structured data in a database
- Use AI to convert natural language into SQL queries
- Display results in an interactive application

---

## 🧠 Key Features

- 📊 Interactive Power BI Dashboard
- 🗄️ Structured data stored in MySQL
- 🤖 Natural Language → SQL using OpenAI
- 🐍 Python backend for query execution
- 🎨 Streamlit-based user interface
- ⚡ Real-time insights generation

---

## 🔄 End-to-End Workflow

1. **Data Preparation & Visualization (Power BI)**
   - Performed data cleaning and preprocessing
   - Built an interactive dashboard
   - Extracted key business insights

2. **Database Management (MySQL)**
   - Stored cleaned data in structured tables
   - Designed schema for efficient querying

3. **Backend Development (Python)**
   - Connected Python to MySQL database
   - Executed dynamic SQL queries
   - Processed and formatted results

4. **AI Integration (OpenAI)**
   - Converted user questions into SQL queries
   - Enabled natural language interaction

5. **Frontend Application (Streamlit)**
   - Built an interactive UI
   - Allowed users to input questions
   - Displayed insights in real-time

---

## ⚙️ Tech Stack

- **Power BI** – Data Visualization
- **MySQL** – Database Management
- **Python** – Backend Logic
- **OpenAI API** – Natural Language Processing
- **Streamlit** – Web Application

---

## 💡 Example Questions

- Total revenue
- Top categories by revenue
- Average rating by category
- Highest revenue city
- Return rate
- Orders by payment method

---

## 🧾 Dataset Features

The dataset includes:

- Product details (category, subcategory, brand)
- Pricing (price, discount, final price)
- Customer metrics (rating, review count)
- Seller performance (seller rating)
- Order details (location, payment method, delivery status)
- Time-based data (purchase date, shipping time)
